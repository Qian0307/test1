✦ 星辰術士・雙月 ✦

我是 雙月 —— 一位在星光與知識之間穿梭的學者型術士。
喜歡探索人類心靈的奧秘， 也熱衷於科技魔法的應用， 也喜歡把所有混亂整理成溫柔的秩序。

目前修行的主法術：

  • 心理學心靈術（NCKU 心理系・情緒 / 發展 / 統計）
  • 資訊安全入門（Web、CTF、漏洞探勘）
  • 程式魔導（JavaScript / Python / 自動化）
  • 觀測術（資料整理、Notion 系統、n8n Automatons）

我的魔力來源是「理解」。 喜歡觀察人、拆解系統、重組資訊， 也擅長把複雜的東西整理得清楚且優雅。

正在用程式語言縫補現實世界的混亂， 也用心理學解讀他人的內在星圖。 希望某一天能創造出屬於自己的「星辰魔法書」—— 一份結合知識、科技與魔法的完整宇宙。
